Reflexão e Aplicação Prática

Propriedades CSS utilizadas

  • display: flex; — Define o container flexível.
  • flex-direction: Controla a direção dos itens (row, column, etc).
  • justify-content: Alinha itens horizontalmente.
  • align-items: Alinha itens verticalmente.
  • flex-wrap: Define se os itens podem quebrar para a próxima linha.
  • order: Controla a ordem dos itens.
  • flex-grow, flex-shrink, flex-basis: Controlam o dimensionamento dos itens.

Aplicação prática

Por exemplo, para criar um menu de navegação responsivo, eu usaria Flexbox para alinhar os itens horizontalmente no desktop e, ao reduzir a largura da tela, permitir que os itens quebrem para uma segunda linha usando flex-wrap. Também usaria justify-content para espaçar os links de forma uniforme e align-items para centralizar verticalmente. Isso facilita muito a adaptação do layout em diferentes tamanhos de tela.
