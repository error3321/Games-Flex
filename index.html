Reflexão e Aplicação Prática

Propriedades CSS utilizadas

  • display: flex; Define o container flexível.
  • flex-direction: Controla a direção dos itens (row, column, etc).
  • justify-content: Alinha itens horizontalmente.
  • align-items: Alinha itens verticalmente.
  • flex-wrap: Define se os itens podem quebrar para a próxima linha.
  • order: Controla a ordem dos itens.
  • flex-grow, flex-shrink, flex-basis: Controlam o dimensionamento dos itens.

Aplicação prática

Em uma galeria de fotos, eu usaria o Flexbox para alinhar as imagens lado a lado em telas grandes, com display: flex e flex-wrap: wrap para permitir que as imagens quebrem linha conforme o espaço disponível. Aplicaria justify-content: center ou space-between para distribuir as fotos de forma equilibrada, e align-items: stretch para manter as alturas consistentes. Isso permite que a galeria se adapte automaticamente em celulares, tablets ou computadores, mantendo o layout organizado e responsivo sem a necessidade de media queries complexas.
